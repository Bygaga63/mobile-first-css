```
Подключить кастомные стили
@font-face {
    font-family: 'Свое название';
    src: url(...)
} 

em - относительная еденица, выщитывается относительно родителя

font-size: 1.2em; - увеличивает размер шрифта на 20% от дефолтового значения. Для шрифтов лучше использовать em

em похож на %, но в margin, line-height, width/height % ведет себя по другому.

rem - задаёт размер относительно размера шрифта элемента <html>

vw, vh, vmin, vmax - относительно экрана
```
